# 디자이너 스튜디오(대시보드) 재설계 시안

## 목표
승인된 디자이너가 **IT를 몰라도** 사진을 올리고, AI 룩을 만들고, 자기 브랜드를 관리할 수 있게.
공개 사이트(K-MODU 미니멀 럭셔리)와 **같은 톤**으로 통일.

## 디자인 원칙
1. 공개 사이트와 동일: **Pretendard**, 블랙/아이보리, 넉넉한 여백, 둥근 pill 버튼.
2. **전부 한국어 + 쉬운 말** (전문용어 제거: Mood/cache/hash/kicker 등 삭제).
3. **한 화면 = 한 가지 일** (인지 부담 최소화).
4. **큰 버튼·큰 터치 영역**, 모바일 우선.
5. **단계 안내(①②③) + 친절한 피드백**("저장됐어요 ✓", "사진을 끌어다 놓으세요").
6. 빈 화면에도 항상 다음 행동 제시.

## 정보 구조 (메뉴)
`홈` · `상품` · `AI 룩` · `숏폼` · `내 브랜드`

```
├─────────┬─────────────────────────────────────┤
│ 🏠 홈    │                                     │
│ 👕 상품   │      (선택한 메뉴 내용)              │
│ ✨ AI 룩  │                                     │
│ 🎬 숏폼   │                                     │

[🏠 홈] [👕 상품] [✨ AI룩] [🎬 숏폼] [🏷 브랜드]
```

```
┌──────────────────────────────────────────┐

│                                          │
│ ┌────────┐ ┌────────┐ ┌────────┐         │
│ │ 상품    │ │ AI 룩  │ │ 숏폼    │         │
│ │  12     │ │   5    │ │   2     │         │
│ └────────┘ └────────┘ └────────┘         │
│                                          │
```

### 2) 상품 올리기 — 가이드형 (가장 중요)
```
┌──────────────────────────────────────────┐
│ ← 상품 올리기                             │
│                                          │
│ ① 사진 올리기                             │
│ ┌──────────────────────────────────┐     │
│ │            📷                     │     │
│ │   사진을 여기로 끌어다 놓거나      │     │
│ │      [ 사진 선택 ]                │     │
│ │   여러 장 한번에 OK · 자동 정리됨  │     │
│ └──────────────────────────────────┘     │
│ [▣][▣][▣] ＋    (올린 사진 미리보기)      │
│                                          │
│ ② 상품 정보                               │
│ 상품 이름  [____________________]         │
│ 종류       [ 상의 ▾ ]                     │
│ 가격       [________] 원                  │
│ 색상       [________]                     │
│ 설명(선택) [____________________]         │
│                                          │
│ ③ [        저장하기        ]              │  ← 가득찬 검정 버튼
│    저장됐어요 ✓                           │
└──────────────────────────────────────────┘
```
- 드래그&드롭 + 클릭 업로드, **여러 장 동시**.
- 업로드 즉시 **자동 최적화(sharp/리사이즈)** → 디자이너는 크기 신경 안 씀.
- 종류는 드롭다운(상의/하의/아우터/가방/신발/액세서리)로 오타 방지.

## 기술 메모 (구현 시)
- 대시보드 전용 스타일을 공개 사이트 토큰(Pretendard, 색/여백)과 통일. 현재 `src/app/globals.css`의 밋밋한 기본값 교체.
- 공통 셸 컴포넌트(상단바 + 사이드/하단 탭) 신설 → 모든 디자이너 페이지가 같은 틀.
- 업로드: 기존 `/api/uploads/product-image` 재사용 + **다중 파일** 지원 + 서버단 sharp 최적화 추가.
- 모든 문구 한국어화, 에러/성공 토스트 한국어.
- 기존 API(`/api/products`, `/api/generate-tryon`)는 그대로 활용 → 백엔드 변경 최소.
- (후순위) "내 공개 카드(포트폴리오 표지+갤러리)" 업로드 → 공개 그리드 DB 연동.

## 단계
1. (현재) 설계안 + 시안 확정
2. 공통 셸 + 홈 화면 구현 → 로컬 시연
3. 상품 올리기/목록(가이드형) 구현 → 시연
4. AI 룩 · 내 브랜드 화면 정리
5. 한국어화·반응형 마감 → 배포
